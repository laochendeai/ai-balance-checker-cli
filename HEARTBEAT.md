# HEARTBEAT.md

## 心跳检查清单（周期性检查）

### 检查频率：每 2-4 小时一次

#### 1. 系统状态检查（每天 1-2 次）
- [ ] 检查 Gateway 状态
- [ ] 检查磁盘空间
- [ ] 检查内存使用

#### 2. 记忆维护（每 2-3 天）
- [ ] 检查 memory/YYYY-MM-DD.md 文件
- [ ] 将重要信息更新到 MEMORY.md
- [ ] 清理过时信息

#### 3. 项目状态（每天 1 次）
- [ ] 检查 git 状态（如果在工作目录）
- [ ] 提交重要的更改

#### 触发提醒（主动联系）
- Gateway 异常
- 磁盘空间不足（< 10%）
- 发现重要邮件或日历事件
- 你 8 小时未联系且出现新情况
