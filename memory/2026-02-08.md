#### 5. 修复多语言切换问题
**问题：**
- 静态文件 404 错误（style.css, i18n.js, app.mjs）
- 语言切换按钮不工作

**原因：**
- vercel.json 配置太复杂
- i18n 使用 fetch 加载翻译文件，导致异步问题
- 事件绑定时机不对

**解决方案：**
- 简化 vercel.json 配置
- 将翻译数据直接内联到 i18n.js 中
- 使用 IIFE 模式避免全局污染
- 正确处理 DOM 加载完成事件
- 删除 locales 目录

**结果：**
- ✅ 静态文件 404 错误修复
- ✅ 语言切换功能正常工作
- ✅ 代码更简洁可靠

#### 6. 新增二维码合并功能 ✅
**需求：**
用户要求添加将多个二维码合并成一个的功能，如将微信和支付宝收款码合并成一个。

**实现方案：**
创建新的 "Merge" 标签页，支持两种合并模式：

**模式 1：选择页模式（适合收款）**
- 生成一个二维码，扫描后跳转到选择页面
- 用户可以选择要使用的二维码（微信/支付宝）
- 实际数据编码在二维码中

**模式 2：网格布局模式（适合分享）**
- 将多个二维码以网格形式排列在一张图中
- 用户可以手动选择要扫描的二维码
- 更直观的视觉呈现

**功能特点：**
- 支持上传 2-4 个二维码图片
- 可为每个二维码添加标签（如"微信支付"、"支付宝"）
- 实时预览合并效果
- 支持下载 SVG 和 PNG 格式
- 完全本地运行，不上传用户数据
- 中英文双语支持

**技术实现：**
- 创建 src/merge.mjs 模块
- 添加 merge 相关 CSS 样式
- 添加 merge 相关 i18n 翻译
- 在 HTML 中添加 Merge 面板和导航按钮
